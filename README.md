# 医学统计与生物信息学习平台

## 📖 平台介绍
这是一个**一站式医学科研学习平台**，整合了医学统计、生物信息学、临床研究、量表评估、公共数据库、科研工具等全流程的科研资源，为医学科研工作者提供全面的学习和工具支持。

平台所有外部链接均为国内可正常访问的权威资源，移除了所有无法访问的维基百科链接，也不使用百度百科链接，确保您可以正常获取所有资源。

---

## ✨ 核心功能
### 1. 完整的统计方法教程
- 从基础的t检验、ANOVA到高级的回归、因果推断，覆盖所有常用统计方法
- 每个方法都提供：
  - 概念简介
  - SPSS操作流程（文字步骤）
  - 带一键复制功能的R代码
  - 带一键复制功能的Python代码
- 所有统计方法的链接均为统计之都的权威教程，国内可正常访问

### 2. 生物信息学分析
- 差异表达分析、功能富集分析、WGCNA、免疫浸润、单细胞测序分析
- 附带完整的可复制的R/Python分析代码，直接用于您的研究

### 3. 常用医学量表汇总
- 20+常用医学量表，覆盖心理健康、生活质量、临床功能、社会支持等
- 每个量表都提供了详细的介绍链接，国内可正常访问
- 完整的量表信效度分析代码，支持Cronbach's α、EFA、CFA等分析

### 4. 传染病数学建模
- 完整的SEIR/SIR/SEIRS/MSEIR四大经典模型教程
- 附带可直接运行的Python实战代码，一键复制即可使用
- 详细的参数说明和建模步骤，新手也能快速上手

### 5. 临床研究资源
- 临床研究设计：RCT、队列、病例对照、横断面研究
- 真实世界研究：PSM、IPTW、DID等混杂控制方法
- 医学机器学习：LASSO、随机森林、XGBoost，以及SHAP可解释性
- 生存分析：Kaplan-Meier、Cox回归、竞争风险模型

### 6. 医学公共数据库
- 全球公共卫生数据库：WHO、CDC、GHDx
- 临床重症数据库：MIMIC-IV、eICU
- 大型人群队列：UK Biobank、NHANES、CHARLS
- 肿瘤数据库：TCGA、GDC、SEER
- 中国本土数据库：CNCB、CNGB、NMDC
- 补充了PMC、ClinicalTrials等重要数据库

### 7. 科研工具合集
- 学术导航、文献下载、医学顶刊直达
- 统计分析软件、在线统计计算工具
- 文献管理、科研作图、投稿选刊神器
- 文献检索数据库

### 8. AI科研工具
- 最新的2025年医学AI工具：MedGemma、Biomni、OpenBioMed
- 分类整理：通用大模型、文献AI、写作润色、统计AI、医学专用AI
- 覆盖科研全流程的AI工具，帮您提升科研效率

## 📱 使用说明
1. **快速跳转**：点击顶部导航栏，可以快速跳转到对应模块
2. **模块展开**：点击大模块的标题（如"医学公共数据库"），可以展开查看详细内容
3. **方法教程**：点击统计方法的"查看详细操作"按钮，可以展开查看完整的教程和代码
4. **代码复制**：所有代码都支持一键复制，点击代码右上角的"复制"按钮即可
5. **外部链接**：所有的外部链接都可以点击打开，都是国内可正常访问的权威资源
