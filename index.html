📊 医学统计学方法

P 值解读

定义：P值：原假设H₀为真时，观察到当前或更极端结果的概率
ASA声明：2016年P值使用规范
常见误区：

  • P<0.05 ≠ 临床意义显著
  • P>0.05 ≠ 无差异，可能样本量不足
  • 多次检验会导致假阳性升高
校正方法： Bonferroni、 FDR、 Benjamini-Hochberg

置信区间 (CI)

定义：置信区间：参数估计的区间范围
优势：同时展示效应大小与精度
解读：

  • 95%CI不含无效值=统计显著
  • 区间越窄，估计精度越高
常用：率差、率比、均数差、风险比

数据前提检验

正态性： Shapiro-Wilk、 KS检验
方差齐性： Levene、 Bartlett
数据类型：

  • 定量：连续、离散
  • 定性：二分类、多分类、有序
  • 生存数据：含时间与截尾
  • 重复测量：多时间点观测

高级统计方法

回归模型： 线性、 Logistic、 Cox
纵向数据： GEE、 线性混合模型
因果推断： PSM、 IPTW、 工具变量、断点回归、双重差分
其他： 中介分析、 多重插补、 竞争风险模型、结构方程模型

基础统计检验

定量数据： t检验、 ANOVA、 非参数检验
定性数据： 卡方检验、 Fisher确切概率、 McNemar检验
相关分析： Pearson、 Spearman、 Kendall相关
一致性： ICC、 Kappa

诊断试验评价

核心指标： 灵敏度、特异度、约登指数
预测值：PPV、NPV、似然比
综合评价： ROC曲线、AUC、校准曲线
报告规范： STARD

📈 传染病数学建模

传染病建模是预测疾病传播、评估防控措施的核心方法，广泛应用于新冠、流感、登革热等传染病研究。

一、核心基础概念

S(t)：易感者 | E(t)：潜伏者 | I(t)：感染者 | R(t)：康复者 | M(t)：死亡者
β：传染率 | σ：潜伏转染率 | γ：康复率 | ξ：免疫丧失率 | μ：死亡率
dX/dt = 人群每日变化量 | R₀=β/γ：基本再生数

二、四大经典模型公式

1. SIR 模型

dS/dt=-βSI/N dI/dt=βSI/N-γI dR/dt=γI

2. SEIR 模型

dS/dt=-βSI/N dE/dt=βSI/N-σE dI/dt=σE-γI dR/dt=γI

3. SEIRS 模型

dS/dt=-βSI/N+ξR dE/dt=βSI/N-σE dI/dt=σE-γI dR/dt=γI-ξR

4. MSEIR 模型

dS/dt=-βSI/N dE/dt=βSI/N-σE dI/dt=σE-(γ+μ)I dR/dt=γI dM/dt=μI

三、参数速查表

符号名称计算方法权威参考
β传染率接触率×感染概率CDC参数标准
σ潜伏率1/潜伏期临床流行病学数据
γ康复率1/传染期疾病自然史
R₀再生数β/γWHO官方定义

四、建模步骤（实战流程）

1. 确定疾病类型与人群特征 → 2. 选择合适的动力学模型 → 3. 拟合/设置参数 → 4. 微分方程求解 → 5. 可视化与结果解读 → 6. 防控措施模拟

五、Python 实战代码（一键复制）

依赖安装：pip install numpy scipy matplotlib


# SEIR传染病模型完整实战代码
import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

# 基础参数
N = 10000  # 总人口
E0,I0,R0 = 0,1,0  # 初始状态
S0 = N - E0 - I0 - R0
beta,sigma,gamma = 0.3,1/5,1/10
t = np.linspace(0,150,150)  # 模拟150天

# SEIR微分方程
def seir(y,t,N,beta,sigma,gamma):
    S,E,I,R = y
    dSdt = -beta * S * I / N
    dEdt = beta * S * I / N - sigma * E
    dIdt = sigma * E - gamma * I
    dRdt = gamma * I
    return dSdt,dEdt,dIdt,dRdt

# 求解并绘图
result = odeint(seir, [S0,E0,I0,R0], t, args=(N,beta,sigma,gamma))
S,E,I,R = result.T
plt.plot(t,S,label='易感者')
plt.plot(t,E,label='潜伏者')
plt.plot(t,I,label='感染者',c='red')
plt.plot(t,R,label='康复者')
plt.xlabel('天数');plt.ylabel('人数')
plt.title('SEIR传染病模型模拟曲线')
plt.legend();plt.show()

六、常见问题与解决方案

✅ 模型不收敛：调整参数范围/优化求解器
✅ 结果不符合实际：重新校准参数
✅ 无真实数据：使用文献报道的R₀值

📝 临床研究设计

证据等级：RCT > 队列 > 病例对照 > 横断面

观察性研究：
• 横断面：患病率调查
• 病例对照：回顾性，计算OR
• 队列：前瞻性，计算RR，遵循STROBE

实验性研究：
• RCT：遵循CONSORT
• 诊断试验：遵循STARD
量表研究：遵循COSMIN

🏥 真实世界研究 (RWS)

定义：真实世界研究：基于常规诊疗数据的临床研究
数据来源：EHR、医保、登记库、区域健康数据

混杂控制： PSM、 IPTW、 DID、 工具变量、高维倾向得分
报告规范： RECORD
优势：外推性强、样本量大、覆盖广泛人群

📋 量表信效度分析

信度： Cronbach's α、重测信度、分半信度
效度：内容效度、结构效度、效标效度
结构分析： EFA、 CFA
规范： COSMIN

⏳ 生存分析方法

基础方法： Kaplan-Meier曲线、 Log-rank检验
多因素： Cox比例风险回归
前提检验：比例风险假设PH检验
拓展：竞争风险模型、时变协变量模型
